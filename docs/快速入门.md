# 快速入门

## 项目简介

TileLang-Ascend 是适配华为 Ascend 昇腾芯片的 TileLang 领域特定语言版本，提供面向 Ascend NPU 的高性能算子开发能力，支持 Ascend Atlas A2/A3等系列产品，提供 Expert 和 Developer 两种编程模式，兼顾极致性能与开发效率。

## 环境要求

### 硬件要求

支持的操作系统：Linux（aarch64/x86_64）  
支持的 Ascend 产品：Atlas A2/A3 系列  
最小硬件配置：单卡 32GB 显存

### 软件依赖

Python（**py3.9-py3.11**），CANN_TOOLKIT，CANN_KERNEL，以及 requirements.txt 等。

CANN 的安装配置脚本详细参考 [安装指南](./安装指南.md)。快捷安装命令参考如下：

```bash
chmod 755 Ascend-cann-toolkit_{version}_linux-{arch}.run
chmod 755 Ascend-cann-kernels-{chip_type}_{version}_linux-{arch}.run

sudo ./Ascend-cann-toolkit_{version}_linux-{arch}.run --install
sudo ./Ascend-cann-kernels-{chip_type}_{version}_linux-{arch}.run --install
```

注意：CANN_TOOLKIT、CANN_KERNEL 是使能昇腾算力卡的关键工具包，需要根据您实际使用的昇腾卡型号选择对应的配套版本（建议 8.5.0 版本），并且安装 CANN 的时间预计耗时 5-10 分钟，请耐心等待安装完成。

requirements 的安装可以参考如下：

```bash
pip install -r requirements.txt
```

## 环境搭建

用户可根据 [安装指南](./安装指南.md) 的环境准备章节步骤搭建 TileLang-Ascend 环境。

### TileLang-Ascend 软件包获取

TileLang-Ascend 当前需通过源码编译安装，暂无 pip 预构建包。

克隆代码后执行一键安装：

```bash
git clone https://github.com/tile-ai/tilelang-ascend.git --recursive -b npuir
cd tilelang-ascend
bash install_npuir.sh
source ~/.bashrc
```

详细安装步骤（含分步手动安装、依赖说明等）请参阅 [安装指南](./安装指南.md)。

## 运行 TileLang 示例

运行示例：vec_add_1d.py（向量加法）

```bash
# 设置 CANN 环境变量（以 root 用户默认安装路径 `/usr/local/Ascend` 为例）
source /usr/local/Ascend/ascend-toolkit/set_env.sh

# 运行示例
python examples/elementwise/vec_add_1d.py
```
